# Bot Analytics Dashboard

Одностраничный сайт-дашборд для аналитики рассылочного бота.

## Что внутри

- количество отправленных сообщений;
- количество живых аккаунтов;
- количество чатов в работе;
- успешность доставки;
- график отправок и доставок;
- статусы чатов;
- состояние аккаунтов;
- таблица кампаний с поиском.

## Запуск

```bash
npm install
npm run dev
```

После запуска Vite покажет локальный адрес, обычно:

```bash
http://localhost:5173
```

## Где менять данные

Все моковые данные находятся в `src/App.jsx`:

- `stats`
- `deliveryData`
- `chatsByStatus`
- `accountGroups`
- `campaigns`

Их можно заменить на данные с API.
